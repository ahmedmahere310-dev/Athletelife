# Athlete LifeOS 🏋️‍♂️

**Athlete LifeOS** is a premium, all-in-one athletic planning ecosystem designed for peak performance. It streamlines your workout programming, nutrition tracking, and daily organization into a single, cohesive interface.

## 🚀 Features

- **Custom Workout Planner**: Generate and customize professional training splits (Arnold, PPL, Upper/Lower) with dynamic exercise management.
- **Nutrition & Macros Dashboard**: Calculate and track your caloric needs and macronutrient distribution based on your physical goals (Bulking, Cutting, or Maintenance).
- **Google Tasks Sync**: Seamlessly synchronize your training sessions and fitness goals directly to your Google Tasks and Google Calendar for maximum accountability.
- **Bilingual Interface**: Full support for both **English** and **Arabic**, including Right-to-Left (RTL) layout optimization.
- **SEO & Performance Optimized**: Includes built-in sitemaps, robots.txt, and optimized meta tags for high-visibility indexing.
- **Modern UI/UX**: Built with a sleek, dark-themed aesthetic utilizing Framer Motion for smooth transitions and Lucide icons for clarity.

## 🛠 Tech Stack

- **Frontend**: [React 19](https://reactjs.org/) + [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [motion/react](https://motion.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Backend/Auth**: [Firebase](https://firebase.google.com/)
- **AI Integration**: [Google Gemini API](https://ai.google.dev/) (Ready for future intelligent features)

## 📦 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/athlete-lifeos.git
   cd athlete-lifeos
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Environment Setup**:
   Create a `.env` file in the root directory and add your credentials (refer to `.env.example`):
   ```env
   VITE_FIREBASE_API_KEY=your_key
   GEMINI_API_KEY=your_key
   ```

4. **Run the development server**:
   ```bash
   npm run dev
   ```

5. **Build for production**:
   ```bash
   npm run build
   ```
